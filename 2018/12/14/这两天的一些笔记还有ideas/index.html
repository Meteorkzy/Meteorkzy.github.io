Git

很久以前就了解过Git,但一直搞不懂这到底是个啥玩意，一直没去探究明白。怎么说，想进入IT行业，自己琢磨的能力一定要够强，这次有了机会，索性把这相关的都搞懂了，不知道是不是每个入门的人都会经历这个过程，还是只是因为我比较蠢。。
好吧，下面开始。

Git是一个软件,这个软件的作用就是给你的项目不同的版本进行管理.

安装Git

自行安装，一路默认，完成后ok。

创建或者修改一个版本库

  • git init
    在使用该命令前，先用下面两个语句声明我的信息。 用到命令git init
    该命令可以理解为告诉Git在这里我要创建一个仓库，然后Git就会生成一个.git文件，该文件是隐藏的，里面就是自己的项目版本仓库。
  • git add
    该命令用于把文件放到版本库中（staged）。
  • git commit
    该命令用于提交改变。 -m后面引号内容是此次提交的备注信息。
